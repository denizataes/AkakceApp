# Akakce App with SwiftUI 🔵⚪️

## Implemented Using 🔥

* MVVM with SwiftUI
* Kingfisher
* ScrollView
* TabView
* NavigationView


## App Features🔩

* You can pull items via API.

* Detailed information can be viewed by clicking on the displayed items.

* Akakce application with pagination structure, loading structure and more..
